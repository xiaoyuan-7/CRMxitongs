一、系统概述

1.1 系统定位

轻量级本地客户管理系统，专为营销团队设计。采用 React + Node.js + SQLite 技术栈，适合轻量级本地部署，无需云端依赖。

1.2 核心价值

  • 轻量部署：SQLite 数据库，无需单独数据库服务器
  • 功能完整：覆盖客户管理、营销跟进、智能提醒全流程
  • 本地安全：数据存储在本地，无需担心云端泄露
  • 易于扩展：模块化设计，便于二次开发

1.3 技术栈

层级技术版本
前端React18
前端 UIAnt Design5
图表库Recharts2.x
路由React Router6
后端Node.js14+
Web 框架Express4
数据库SQLite3
认证JWT + bcrypt-

二、核心功能模块

2.1 营销任务管理

功能描述：营销目标设定与进度追踪

  • 任务目标设定：可设置有效户、高质量客户的目标数量
  • 进度追踪：实时显示完成进度（圆形进度条可视化）
  • 储备客户管理：区分已完成和储备中的客户
  • 落地预测：本月落地、三个月落地统计

2.2 企业管理

功能描述：企业客户信息全生命周期管理

  • 企业列表：展示企业名称、关键人、开户状态、代发状态、有效户、高质量、进度状态
  • 企业详情：企业简介、财务信息、行业、上下游信息
  • 搜索筛选：支持按名称、状态等多维度搜索
  • 数据导出：CSV 格式导出

2.3 关键人管理

功能描述：企业关键联系人详细信息管理

  • 多关键人支持：每个企业可添加多个关键人
  • 详细信息：出生年月、职位、家庭信息、喜好、推荐送礼
  • 主要联系人标记：可设置一个主要联系人
  • 生日自动计算：年龄、生肖、下次生日提醒

2.4 每周计划

功能描述：营销任务周计划安排与执行

  • 7 天视图：周一至周日任务安排
  • 任务分配：可为每天分配企业跟进任务
  • 时间段标记：上午/下午/晚上
  • 优先级设置：高/中/低优先级
  • 联系频次关联：本周触达、两周触达、本月触达、一月前触达

2.5 智能提醒

功能描述：自动化提醒生成与管理

  • 生日提醒：自动生成本年生日提醒
  • 节假日送礼提醒：自动生成节假日送礼提醒
  • 自定义提醒：手动添加提醒
  • 今日提醒/即将到期提醒

2.6 数据报表

功能描述：营销数据可视化分析

  • 转化率分析：开户→代发→有效户→高质量转化漏斗
  • 跟进趋势图：30 天跟进记录统计
  • 业绩趋势图：6 个月业绩走势
  • 进度状态分布：各阶段企业数量分布
  • 行业分布：企业行业分类统计

2.7 用户权限

功能描述：用户认证与权限管理

  • 用户注册/登录：JWT 认证
  • 角色管理：管理员/普通用户
  • 权限控制：管理员可管理用户角色

五、部署指南

5.1 环境要求

软件最低版本推荐版本
Node.js14.0.018.x LTS
npm6.0.09.x

5.2 开发环境部署

# 1. 安装后端依赖
cd /root/.openclaw/workspace/crm-system/backend
npm install

# 2. 安装前端依赖
cd /root/.openclaw/workspace/crm-system/frontend
npm install

# 3. 启动后端服务
cd /root/.openclaw/workspace/crm-system/backend
npm start

# 4. 启动前端开发服务器
cd /root/.openclaw/workspace/crm-system/frontend
npm start

5.3 生产环境部署

# 1. 构建前端
cd frontend && npm run build

# 2. 启动后端（提供静态文件）
cd backend && npm start

# 访问 http://localhost:3001

六、安全说明

6.1 已实现的安全措施

  • ✅ 密码加密：bcrypt 哈希存储
  • ✅ JWT 认证：Token 验证登录状态
  • ✅ 输入验证：请求参数校验
  • ✅ SQL 注入防护：参数化查询
  • ✅ CORS 配置：跨域请求控制
  • ✅ 权限控制：角色权限分离

6.2 生产环境建议

  • ⚠️ 修改 JWT_SECRET 为强随机密钥
  • ⚠️ 启用 HTTPS
  • ⚠️ 配置防火墙规则
  • ⚠️ 定期备份数据库文件
