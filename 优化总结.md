## 二、功能优化

### 1. 业务状态更新
- **旧：** 审核中、已成交、无效
- **新：** 📞 跟进中、✅ 已落地、❌ 无效

### 2. 条线选项更新
- **旧：** 公司业务部、零售业务部、金融市场部、投资银行部、资产管理部

### 4. 线索挖掘增强
- ✅ 批量导入（支持 Excel 复制粘贴）
- ✅ 批量编辑
- ✅ 板块管理
- ✅ 线索状态跟踪

### 5. 导出格式优化
- **旧：** TSV 格式（分列问题）
- **新：** HTML 表格格式（Excel 自动识别）

## 七、使用说明

### 协同融合模块
1. 点击"协同融合"标签
2. 点击"+ 新建转介"填写信息
3. 状态改为"已落地"自动积 1 分
4. 查看图表和排行榜

### 线索挖掘模块
1. 点击"线索挖掘"标签
2. 点击"+ 新建线索板块"
3. 点击板块进入详情
4. 可批量导入/编辑线索

### 数据导出
1. 在企业列表勾选企业
2. 点击"📤 导出"
3. 选择字段
4. 下载 HTML 格式文件
